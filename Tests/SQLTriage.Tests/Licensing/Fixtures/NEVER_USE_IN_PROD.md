# WARNING: TEST FIXTURES ONLY

The keys and bundles generated by `BundleFixtureFactory` are **test fixtures**.

- `TestKey` (0x01..0x20) is a known, publicly visible value committed to the repository.
- `FreeKey` (all-zero bytes) is the well-known Free-tier key baked into the app.

**NEVER use these keys to encrypt real customer bundles.**

Real customer keys are generated by the CorpusEncryptor (`--gen-key` mode) and are unique per customer. They are never committed to any repository.
